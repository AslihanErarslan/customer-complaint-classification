# customer-complaint-classification
## Proje Dosyaları
Bu repoda projede kullanılan veri setine erişim bağlantısı yer almaktadır.

## Veri Seti
Müşteri şikayet metinlerinden oluşan veri seti, herhangi bir ön işleme uygulanmadan orijinal haliyle paylaşılmıştır. Veri seti finansal ürün ve hizmetlere yönelik gerçek müşteri şikayetlerini içermektedir ve metin sınıflandırma çalışmaları için kullanılmaktadır.

GitHub dosya boyutu sınırlamaları nedeniyle veri seti harici olarak barındırılmaktadır.

## Bilgilendirme
Veri seti Kaggle platformu üzerinden paylaşılmış olup farklı kategorilerde müşteri şikayet metinleri içermektedir. Proje kapsamında modeller, veri setinin indirilen orijinal hali kullanılarak geliştirilmiştir.
